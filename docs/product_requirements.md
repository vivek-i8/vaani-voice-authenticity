# VAANI — Product Requirements Document

## 1. Problem Statement

AI voice cloning tools can now replicate human voices with high realism.  
These technologies are increasingly used in scam calls, misinformation, and identity fraud.

Detecting whether a voice recording is human or AI-generated has therefore become an important security challenge.

## 2. Product Vision

VAANI aims to provide a system capable of detecting whether a short voice recording is human or AI-generated.

## 3. Target Users

- Security researchers  
- Media verification teams  
- Financial fraud prevention systems  
- General users verifying suspicious voice recordings

## 4. Core Features

- Audio upload and playback
- AI voice detection
- Confidence scoring
- Signal analysis visualization
- Explainability using natural language summaries

## 5. User Workflow

User uploads audio  
↓  
System processes audio  
↓  
Model analyzes speech signals  
↓  
System returns classification result with explanation

## 6. Key Outputs

- Prediction: Human / AI / Inconclusive
- Confidence score
- Signal certainty metrics
- Acoustic feature analysis

## 7. Product Goals

- Detect AI-generated speech reliably
- Provide interpretable results
- Maintain simple user interaction
